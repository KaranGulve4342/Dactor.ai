## Starting the project locally

1. Clone the repository

   ```bash
   git clone https://github.com/
   ```

2. Install dependencies

   ```bash
   cd shadcn-ui-sidebar
   npm install
   ```

3. Run the development server

   ```bash
   npm run dev
   ```
